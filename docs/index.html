想象一下：你家的猫，今天。你早上 8:30 出门。它趴在门口，尾巴微微抖了一下，然后你消失了。

接下来的九个小时，没有发生任何事。它走到窗边，看了一会儿鸽子，走回沙发，睡了。醒来，走到食盆——食盆是满的，但它不饿，它没有花体力。这一天，它的心跳从未超过静息值，肌肉没有收缩一次用于捕猎，神经系统没有被真正激活过。

猫天生是猎手，5000 万年的进化刻在 DNA 里。现代公寓给不了它猎物。结果是 41% 超重，30% 焦虑，平均寿命比野外短 3–5 年——不是因为医疗条件差，是因为它们在最安全的地方慢慢退化了。

传统玩具给的是"刺激"。猫真正需要的是"博弈"——一个会逃跑、会反击、永远无法被完全预测的对手。Hunter 就是这个对手。

02 · Core Value

五大核心卖点

每个卖点对应一个评审维度，每个概念都是全球首创。

01 · BIOLOGICAL LOOP

完整的生物学闭环

猫的捕猎行为分三阶段：寻找 → 追逐 → 捕获。任何阶段缺失，多巴胺回路都不完整。这就是为什么激光玩具会让猫焦虑——它永远无法"抓住"。Hunter 构建了完整的狩猎 → 捕获 → 进食闭环：猫扑中时底盘投出零食。市面上 1000+ SKU，无一做到。

→ 对应评审维度：创新性
02 · SURPRISE ENTROPY

惊喜熵引擎

所有 AI 都在优化——但优化是猫无聊的根源。Hunter 引入新指标：惊喜熵（Surprise Entropy）——让猫始终保持 70% 把握 + 30% 不确定。它会故意失败、故意打破规律、故意"笨拙"。当某个套路被猫识破，立刻主动弃用。这是 Csikszentmihalyi 心流理论在物理交互上的工程实现。

→ 对应评审维度：技术难度 + 创新性
03 · AI FOR THE INARTICULATE

第一个为不会说话的存在设计的 AI

ChatGPT、Siri、Copilot——所有 AI 都依赖语言作为接口。Hunter 读的是尾巴姿态、耳朵角度、瞳孔大小、全身体态。今天做猫，明天平移到：失能老人（2.8 亿）、低龄婴儿、特殊需求儿童。猫是起点，不是终点。

→ 对应评审维度：商业潜力（长期赛道）
04 · PERSONALITY EVOLUTION

每台 Hunter 最终都不一样

传统硬件出厂是什么样，用一年还是什么样。Hunter 出厂时相同，但和"它的那只猫"持续共同进化。六个月后，张三家的 Hunter 和李四家的 Hunter，"灵魂"已经不一样了。用户换设备 = 失去六个月的人格化记忆，Churn 率接近零。

→ 对应评审维度：商业潜力（护城河）
05 · MEMORY BOX

AI 日记与记忆盒子

每晚推送的不是图表，是 LLM 写的文字："今天 Mochi 试图捕捉那只羽毛玩具 23 次，失手后假装根本不在乎。" 73% 的主人最担忧猫独处是否孤独，68% 认为无法记录日常是最大遗憾。Hunter 在猫的整个生命中记录每一天。猫离世后，自动整理「它的一生」。

→ 对应评审维度：现场讲述效果
03 · Features

11 项核心功能

三大模块，感知 → 逗猫 → 照料，完整闭环。点击功能卡片查看 5W1H 详细说明。

P0

👁视觉追踪

YOLOv8-nano 实时定位猫咪，PID 控制底盘自动跟随，确保逗猫工具始终在最佳距离。25fps 推理，延迟 <200ms。

P0

🎙叫声识别

双麦克风阵列波束成形降噪，MFCC + CNN 分类 4 种情绪：饥饿 / 撒娇 / 警戒 / 满足，准确率目标 ≥ 85%。

P0

⚡活跃度感知

Lucas-Kanade 光流 + 时间段加权，每 10 秒输出活跃度分 0–100，自动在疯狂 / 轻柔模式间切换。

P0

🪄逗猫棒挥舞

4-DOF 机械臂，贝塞尔曲线轨迹生成，±15° 随机扰动。每 3 次动作约 1 次让猫"抓到"，维持成就感。

P0

🔴激光点追逐

650nm 5mW Class II 激光，二轴云台控制，随机游走 + 禁忌搜索算法，底盘联动确保光点不消失于阴影。

P0

🐟零食投喂奖励

步进电机 + 螺旋推进器定量出料，扑中后 1 秒内投喂 1–3 粒，每日上限可 APP 设置，构成完整狩猎闭环。

P1

📸高光捕捉

30fps 环形缓冲，萌度评分模型（正脸置信度 + 双眼占比 + 嘴部开合），自动回溯提取最佳帧。

P1 ✅

😹表情包生成

VLM 生成 3 种风格配文（严肃体 / 自恋体 / 茫然体），Pillow 合成黑描边白字，每日上限 20 张推送 APP。

P1

📖AI 日记

SQLite 聚合全日数据，LLM 以猫咪口吻生成 200 字日记，每天 20:00 推送，异常事件即时插播。

P1

💧饮水监测

超声波液位传感器 ±1mm 精度，每 30 秒采样，12 小时无饮水触发预警，机器人主动驶向猫咪提示。

P2

📱远程 APP 控制

WebRTC 实时视频流，MQTT over TLS 加密指令，延迟目标 <500ms，React Native 跨平台，7 日历史回放。
